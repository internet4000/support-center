Find (visit) my Tumblr blog (the blog URL)?

  1. You need to be logged in Tumblr, at https://tumblr.com
  2. Go to https://tumblr.com and click the person icons (üë§)
  3. In the dropdown menu that opened, select (click) your blog

Select and use a Tumblr Theme for my blog

  1. You need to have and account on Tumblr, create it at https://tumblr.com
  2. You need to be logged in Tumblr, log in at https://tumblr.com
  3. Visit the page http://tumblr.internet4000.com
  4. Click the "All themes" link, or select a theme on this page.
  5. On this new page, there is a selection of themes that you can use.
    Click "preview" to see an exemple blog using this Theme
    Click "install", and select on which blog you want to install and use this theme.

Create a new navigation element on my Tumblr site

  1. You need to be logged in Tumblr, at https://tumblr.com
  2. Go to your Tumblr website url (ex: https://my-website.tumblr.com, find it via the link in step before.)
  3. In the top right of your Tumblr blog, click "Edit Appeareace" menu icon, to customize your site.
  4. Scroll to the bottom of the menu on the left.
  5. Now click on "+ Add a page".
  6. You can select between different type of pages that you can create, "standard layout", "custom layout", "redirect"

Read the following answers to find out what are the differences, and how to use, these different pages.
